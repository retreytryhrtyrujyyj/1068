# 基于协同过滤算法电影推荐系统

## 项目简介
基于协同过滤算法的图书推荐系统是基于Java Web，Mysql数据库的电影推荐系统。本系统支持三种协同过滤算法：1.基于用户的协同过滤算法2.基于物品的协同过滤算法3.基于评分的协同过滤算法（Slope One）
